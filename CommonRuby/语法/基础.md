## Ruby笔记

### 学习前基础
1. 输出
    ```
    print("abc")
    print "abc"
    puts("abc")
    puts "this is", "ruby"
    ```
2. 替位符号
    ```
    name = "zjs"
    puts("my name is #{name}")
    ```
3. '' 与 ""的区别

    puts('\n') 输出 \n

    puts("\n") 直接换行

4. p方法，直接输出对象内容

5. 注释
    
    一共有两种方法
    第一种
    ```
    # this is comment
    name = "zjs"
    puts name
    ```
    第二种
    ```
    =begin
    学习记录
    2018/12/3 初次记录
    2018/12/4 追加内容
    =end
    ```
5. 控制语句

    if 语句
    ```
    if 8 > 9
        puts "8 > 9"
    else 
        put "8 <= 9"
    end
    ```

    while语句
    ```
    i = 10
    while i > 0
        puts i, "\n"
        i = i + 1
    end
    ```

    times方法
    ```
    12.times do 
        print("hhh")
    end
    ```
6. 数组
    ```
    names = ["zhu", "jin", "shan"]
    puts names[0]
    names[2] = "j"
    # 数组大小
    puts names.size
    # 数组循环
    names.each do |n|
        puts n
    end
    ```
7. Hash
    ```
    stu = {name: "zjs", age: 24}
    # 另外一种方法
    stu = {name => "zjs", age => 24}
    # 读
    puts stu[:name]
    # 写
    stu[:name] = "zhujs"
    # 循环
    stu.each do |k, v|
        puts "#{k}:#{v}"
    end
    ```
