## 数值类

### 数值类示意图
+ Numberic
    - Integer
        * Fixnum # 常用的就是这个普通的整数
        * Bignum
    - Float
    - Rational # 有理数
    - Complex


1. 有理数构成 `a = Rational(3, 5)` 代表 3 / 5
2. 复数： Complex(实数, 虚数)

### 数值的字面量
```
123     # 10进制整数
0123    # 8进制123， 10进制为83
0o123   # 8进制123
0d123   # 10进制123
0x123   # 16进制123
0b1110100 # 二进制
123.45  # 浮点数
12.3e4  # 浮点数指数表示法
123_456 # 10进制数123456
```
